# Kueski Pay Addon — Overview

**Versión:** 1.11 — **Fecha:** 2026-03-02  
**Responsables:** Víctor / Carlos  
**Plataforma:** CS-Cart 4.19.1.SP2  
**Entorno activo:** DEV — `https://dev.importacionesamexico.com.mx`

---

## Objetivo

Integrar Kueski Pay como método de pago en CS-Cart, permitiendo a los clientes 
financiar sus compras directamente desde el checkout. La integración cubre el 
flujo completo: creación de pago, redirección al widget de Kueski, notificación 
vía webhook y actualización automática del estado de la orden.

## Flujo general
```
Cliente selecciona Kueski Pay
        ↓
kueski_pay.php → POST /v1/payments → Kueski API
        ↓
Redirect → Widget Kueski (sandbox-ui / producción)
        ↓
Cliente autoriza / cancela / es rechazado
        ↓
Kueski → POST webhook → kueski.php
        ↓
Guard + Idempotencia + fn_change_order_status()
        ↓
Orden actualizada en CS-Cart
```
