## 🔐 Principios Técnicos

<Callout type="info">
El backend es la fuente de verdad.  
Ningún estado crítico se define desde la UI.
</Callout>

- Webhooks idempotentes
- Validación obligatoria antes de confirmar órdenes
- Separación estricta DEV / STAGING / PROD
- Logging sin exponer secretos
- Diseño con rollback seguro
